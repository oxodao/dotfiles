# Oxodao's ArchLinux Rice (A.K.A dotfiles)
Here are my config files for my desktop environment

## Pre-requisites
You need to have installed theses software:
	curl, git, tmux, termite, neovim, i3wm-gaps, i3blocks, zsh

Maybe others that I don't remember. Check the logs.

## How to install
Just clone the repository where you wan't and have at least read access (Should not require write access though I have no clue if it will)
Then cd into it and ./install.sh

Tested on ArchLinux and Neovim. Probably needs a bit of workaround to work under osx &| standard vim
